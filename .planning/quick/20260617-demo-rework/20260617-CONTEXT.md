# Quick Task 20260617: Demo Scene Rework — Context

**Gathered:** 2026-06-17
**Status:** Ready for planning

## Task Boundary

Rework `demos/demo.py` and `scenes/suturing_demo.json` so the scene is a
realistic suturing demo with:

1. **Soft, deformable tissue** (not rigid boxes) — the two skin patches
   should bend under load (needle pressure, gripper pinch) instead of
   acting as immovable blocks.
2. **Realistic needle size** — the current 2×2×10 cm box is a
   thumb-sized club. A real curved suturing needle is ~16 mm long with
   a ~1.2 mm wire — 3 orders of magnitude smaller in cross-section.
3. **Visible gripper at the end of the arm** — the chain has a
   `<joint name="..._gripper">` slide actuator but no actual jaw
   geometry. The viewer should see a small needle driver or grasper
   attached to the last link.

## Implementation Decisions

### Soft tissue
- Use `<flexcomp type="grid">` in MuJoCo — built into scene_builder's
  existing soft-body code path. Switches the two tissue boxes from
  rigid `<geom>` to FEM mesh.
- One tissue per flexcomp grid (keep them as 2 separate skin patches,
  not 1 merged surface) so the suturing task — needle through both
  patches — still makes sense.
- Reasonable FEM defaults: stiffness ~5000 N/m, density 1100 kg/m³
  (matches the existing scene JSON), Poisson's ratio 0.45.
- Test resolution: low (~5×5×2) to keep sim fast. Surgical RL doesn't
  need fine tissue meshes.

### Needle size
- Change instrument type from `"custom"` to `"needle"` in the scene
  JSON. The `_needle` generator (mesh_generator.py:166) already produces
  a realistic torus with 8mm major radius and 0.6mm wire radius.
- The primitive fallback path will pick this up automatically once the
  type matches the registered generator.
- Place the needle in the workspace near the tissue, with its tip
  pointing toward the right skin patch (the standard suturing starting
  position).

### Gripper
- Add two short cylindrical "jaws" (3 mm radius, 15 mm long) at the
  end of the last link, offset symmetrically in y so they form a V
  shape with a small opening.
- The existing `surgical_arm_1_gripper` slide joint (0..0.05 m)
  controls jaw spacing — visually you can see the jaws open and close
  with the slide position.
- Color: silver/gray to match a metal surgical instrument.
- Implemented as two `<geom>` children of the last link body (no new
  body — the slide joint already controls gripper position).

### Where the changes go
- `scenes/suturing_demo.json` — switch tissue to soft, fix needle type,
  resize needle, recenter the needle.
- `src/surg_rl/simulators/scene_builder.py` — add gripper jaw geoms
  in `_add_robot_to_mjcf` (the `_gripper` slide joint is already
  added; just add visual geoms).
- `demos/demo.py` — no functional changes; the scene JSON change is
  picked up automatically. May need a banner update describing the
  new demo.
- `tests/test_scene_builder.py` — new tests for soft tissue rendering,
  realistic needle mesh fallback, and gripper jaw geoms.

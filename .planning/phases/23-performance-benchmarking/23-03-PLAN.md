- Input: aggregated_results filtered for this backend
- Creates figure with 2 subplots (or single with dual y-axes):
  - **Subplot A (mean±1σ)**: x=timestep, y=mean reward, fill_between(mean-std, mean+std, alpha=0.2)
  - **Subplot B (IQM+CI)**: x=timestep, y=IQM, fill_between(ci_low, ci_high, alpha=0.3)
- Or single plot with both: mean±1σ as light shade, IQM+CI as darker line with CI band
- Per D-08: both statistical aggregations shown — IQM for publication, mean±1σ for quick inspection
- X-axis: timesteps (log scale optional via config)
- Y-axis: episodic return (cumulative reward)

- Title: f"Learning Curves — {config.task} — {backend.upper()}"
- Save: `{output_dir}/{backend}/learning_curve.png`
- DPI: 300, bbox_inches='tight'

**Method 2**: `render_success_rate_bars(backend: str) -> Path`
- Bar chart: x=algorithms, y=success_rate (0-1)
- Error bars: rliable stratified bootstrap CI (from learning_curve_iqm or scalar_metrics)
- Significance annotations: use rliable `probability_of_improvement` or `credible_interval` to annotate pairwise comparisons (e.g., "*" for p<0.05 vs best)
- Per D-07: bar charts with rliable statistical significance
- Color: algorithm palette, DreamerV3 hatched if pending
- Title: f"Success Rate — {config.task} — {backend.upper()}"
- Save: `{output_dir}/{backend}/success_rate_bars.png`

**Method 3**: `render_results_table(backend: str) -> Path`
- Matplotlib table plot (not DataFrame.to_html — actual rendered table image)
- Rows: algorithms (+ DreamerV3 if present)
- Columns: Mean Reward ± Std, Success Rate (%), Mean Episode Length, Wall-Clock Time (s), Sample Efficiency (reward/step)
- Values from scalar_metrics per algorithm
- Format: mean ± std for reward, percentage for success rate, 1 decimal for others
- Title: f"Results Summary — {config.task} — {backend.upper()}"
- Save: `{output_dir}/{backend}/results_table.png`

**Method 4**: `render_all() -> list[Path]`
- For each backend in config.backends (expanded):
  - Call all three render methods
  - Return list of generated plot paths
- If config.render_plots is False, return empty list

|----------|-------------|
| Test mocks → subprocess.run | Mocks intercept system calls; no real binaries invoked in unit tests |
| Docker build → host kernel | Containerized; no direct host access beyond --gpus or --device mounts |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-08 | Tampering | Dockerfile base images | mitigate | Pin exact image tags (nvidia/cuda:12.2.0-runtime-ubuntu22.04, rocm/dev-ubuntu-22.04) instead of floating latest |
| T-06-09 | Denial of Service | test mocks | accept | Mocks are test-only; production code uses real subprocess |
| T-06-10 | Information Disclosure | test output | accept | Test output contains mocked GPU versions, not real system data |
</threat_model>

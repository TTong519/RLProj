### Task 2: Create `surg_rl/rl/rllib/env_wrapper.py`
**Action:** Create `env_creator` factory and registration helper.

**Pattern:**
```python
from __future__ import annotations

from typing import Any

from ray.tune.registry import register_env

from surg_rl.rl.environment import SurgicalEnv, SurgicalEnvConfig


def make_surgical_env(env_config: dict[str, Any] | None = None) -> SurgicalEnv:
    """Create a SurgicalEnv from an RLlib env_config dict.

    RLlib passes env_config as a plain dict. We convert it to
    SurgicalEnvConfig and force render_mode=None (rendering crashes
    inside Ray worker processes).
    """
    env_config = env_config or {}
    # Force headless — Phase 7 RenderThread crashes in Ray workers
    env_config["render_mode"] = None
    cfg = SurgicalEnvConfig(**env_config)
    return SurgicalEnv(cfg)


def register_surgical_env(name: str = "surg-rl") -> None:
    """Register SurgicalEnv with RLlib's tune registry."""
    register_env(name, make_surgical_env)
```

**Critical constraint:** Do NOT pass the raw `env_config` dict straight into `SurgicalEnv`. Always convert via `SurgicalEnvConfig(**env_config)`. This avoids `TypeError: got unexpected keyword argument 'scene_path'`.

**Pitfall guard:** `SurgicalEnvConfig` contains nested dataclasses (`RewardConfig`, `ObservationConfig`, `ActionConfig`). If these are passed as nested dicts in `env_config`, `SurgicalEnvConfig` (plain dataclass, not Pydantic) will NOT validate them. We must accept that RLlib `env_config` should only contain primitive fields that `SurgicalEnvConfig` accepts directly (str, int, float, bool, None). Nested config objects must be pre-instantiated by the caller or omitted.

**Verification:**
```python
env = make_surgical_env({"scene_path": "scenes/simple_suturing.json", "simulator_type": "mujoco"})
assert env.config.scene_path == "scenes/simple_suturing.json"
assert env.render_mode is None  # forced headless
```

### Task 3: Create `surg_rl/rl/rllib/config.py`
**Action:** Create `RllibConfig` dataclass that converts from `TrainingConfig` / `AlgorithmConfig` to RLlib `PPOConfig` / `SACConfig`.

**Fields to map:**
| RLlibConfig field | Source | RLlib API |
|---|---|---|
| `env_name` | default `"surg-rl"` | `.environment(name)` |
| `env_config` | `TrainingConfig` → dict | `.environment(env_config=...)` |
| `algorithm` | `"PPO"` or `"SAC"` | `PPOConfig` / `SACConfig` class |
| `framework` | always `"torch"` | `.framework("torch")` |
| `num_env_runners` | `TrainingConfig.n_envs - 1` | `.env_runners(num_env_runners=...)` |
| `num_learners` | auto from GPU count | `.learners(num_learners=...)` |
| `num_gpus_per_learner` | 1 if GPU else 0 | `.learners(num_gpus_per_learner=...)` |
| `train_batch_size_per_learner` | `AlgorithmConfig.n_steps * n_envs` | `.training(train_batch_size_per_learner=...)` |
| `lr` | `AlgorithmConfig.learning_rate` | `.training(lr=...)` |
| `gamma` | `AlgorithmConfig.gamma` | `.training(gamma=...)` |
| `lambda_` | `AlgorithmConfig.gae_lambda` (PPO only) | `.training(lambda_=...)` |
| `clip_param` | `AlgorithmConfig.clip_range` (PPO only) | `.training(clip_param=...)` |
| `num_epochs` | `AlgorithmConfig.n_epochs` (PPO only) | `.training(num_epochs=...)` |
| `entropy_coeff` | `AlgorithmConfig.ent_coef` (PPO only) | `.training(entropy_coeff=...)` |
| `vf_loss_coeff` | `AlgorithmConfig.vf_coef` (PPO only) | `.training(vf_loss_coeff=...)` |
| `total_timesteps` | `TrainingConfig.total_timesteps` | Used for stop criteria |
| `seed` | `TrainingConfig.seed` | `.environment(env_config={"seed": ...})` |
| `save_dir` | `TrainingConfig.log_dir` | Checkpoint path |
| `checkpoint_freq` | `TrainingConfig.save_freq` | Save every N timesteps |

**GPU auto-detection logic:**
```python
import torch

gpu_count = torch.cuda.device_count()
if gpu_count >= 2:
    num_learners = gpu_count
    num_gpus_per_learner = 1
elif gpu_count == 1:
    # Single GPU: local learner for better throughput
    num_learners = 0
    num_gpus_per_learner = 1
else:
    # CPU-only
    num_learners = 0
    num_gpus_per_learner = 0
```

**Build algorithm config method:**
```python
def build_rllib_config(self) -> "PPOConfig | SACConfig":
    from ray.rllib.algorithms.ppo import PPOConfig
    from ray.rllib.algorithms.sac import SACConfig
    
    algo_cls = PPOConfig if self.algorithm == "PPO" else SACConfig
    
    config = (
        algo_cls()
        .environment(self.env_name, env_config=self.env_config)
        .framework("torch")
        .env_runners(num_env_runners=max(0, self.num_env_runners))
        .learners(
            num_learners=self.num_learners,
            num_gpus_per_learner=self.num_gpus_per_learner,
        )
    )
    
    # Training hyperparameters
    training_kwargs = {
        "lr": self.lr,
        "gamma": self.gamma,
        "train_batch_size_per_learner": self.train_batch_size_per_learner,
    }
    if self.algorithm == "PPO":
        training_kwargs.update({
            "lambda_": self.lambda_,
            "clip_param": self.clip_param,
            "num_epochs": self.num_epochs,
            "entropy_coeff": self.entropy_coeff,
            "vf_loss_coeff": self.vf_loss_coeff,
        })
    
    config = config.training(**training_kwargs)
    return config
```

**Verification:** `RllibConfig.from_training_config(training_cfg).build_rllib_config()` returns a `PPOConfig` instance with correct hyperparameters.

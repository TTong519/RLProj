## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Test runner → test module | pytest collection executes the module-level pytestmark skipif; missing torch/jax must not crash collection |

| Subprocess → parent | The _JsonStdout wrapper (subprocess.py:23) is the only communication channel; if it fails, the parent's recv() raises BlockingIOError or blocks forever |

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Test runner → test module | pytest collection executes the module-level pytestmark skipif; missing torch/jax must not crash collection |
| Test method → production code | run_dreamer_training spawns a real subprocess via multiprocessing.Pipe; if the pipe is closed prematurely, the parent blocks indefinitely |
| Subprocess → parent | The _JsonStdout wrapper (subprocess.py:23) is the only communication channel; if it fails, the parent's recv() raises BlockingIOError or blocks forever |

## STRIDE Threat Register

| T-30-04 | Elevation of Privilege | subprocess spawn | accept | The test spawns a daemon subprocess via multiprocessing; the subprocess is sandboxed to the test's process tree and is terminated in the finally block of run_dreamer_training (training.py:360 `subprocess.shutdown()`).

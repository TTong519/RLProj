    - SIGINT handler registered on main thread; calls close() then re-raises KeyboardInterrupt
    - Falls back silently if not on main thread (e.g., SubprocVecEnv worker)
    - atexit.register(self.close) added in __init__
    - Guard against double-registration if env created multiple times
  </behavior>
  <action>
    In SurgicalEnv.__init__, add after existing init code:
    ```python
    self._setup_signal_handlers()
    ```

    Add method:
    ```python
    def _setup_signal_handlers(self) -> None:
        import signal
        import atexit

        if getattr(self, "_handlers_registered", False):
            return

        def _handle_sigint(signum, frame):
            self.close()
            raise KeyboardInterrupt

        try:
            signal.signal(signal.SIGINT, _handle_sigint)
        except ValueError:
            pass  # Not on main thread (e.g., SubprocVecEnv)

        atexit.register(self.close)
        self._handlers_registered = True
    ```
  </action>
